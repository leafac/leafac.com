1 Setup

The website is a static webpage generated by Pollen, a Racket library. Run the following to install the Racket dependencies:

$ raco pkg install --name www-leafac-com $(pwd)

The only non-Racket dependency is the Python library Pygments, for syntax highlighting. Install it by running the following:

$ pip install pygments

Start the preview server with:

$ make
